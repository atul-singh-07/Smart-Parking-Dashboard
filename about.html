IoT · Firebase · Real-Time

Built to Solve Real Parking
Problems

SmartPark is an IoT-based parking management system using ESP32, ultrasonic sensors, and Firebase Realtime Database — providing live slot monitoring, pre-booking, and smart analytics for modern urban parking.

🚗

Real-Time Monitoring

Ultrasonic sensors track vehicle presence and push updates to Firebase instantly.

📡

IoT Sensors

ESP32 microcontrollers communicate with the web platform over WiFi with zero lag.

🌐

Live Web Dashboard

Firebase onValue listeners keep the UI synced in real-time — no page refresh needed.

⚡

Smart Booking

Atomic writes prevent double-booking, and auto-assign gives you the nearest free slot.
